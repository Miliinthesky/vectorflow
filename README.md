# VectorFlow AI - Stop tracing. Start creating.

🌐 **English** | [简体中文](README_zh.md)

### AI-powered vector production workflow for Adobe Illustrator.

*More than vectorization. Automate your entire Illustrator workflow.*

VectorFlow AI is an AI-powered desktop application that automates the entire vector production workflows in Adobe Illustrator.

It analyzes your images, generates editable Adobe Illustrator (.ai) files, classifies output quality, and streamlines repetitive production tasks for designers and manufacturers.

# Requirements

Before using VectorFlow AI, please make sure your computer meets the following requirements.

✅ Adobe Illustrator (Required)

✅ Gemini API Key

✅ macOS

No manual interaction with Adobe Illustrator is required.

VectorFlow AI automatically launches and controls Illustrator during the production process.

# How it works?

When launching VectorFlow for the first time, a welcome window will appear automatically. Please enter your Gemini API Key.

<img width="448" height="232" alt="Welcome En:CN" src="https://github.com/user-attachments/assets/1948be9f-d085-4bc6-a462-a9e747f0c279" />

Place the PNG or JPG images to be processed into a folder, then select the input folder and output folder. Click **"Start Smart Processing"**. Adobe Illustrator will launch automatically and complete the remaining processing without any manual intervention.

<img width="898" height="681" alt="UI En" src="https://github.com/user-attachments/assets/95c131c7-fb7a-4406-b349-d57363cd5304" />

Click **"Settings"** to adjust the output size. The aspect ratio will be automatically maintained based on the original image. Choose either the Chinese or English interface, then click **"Save"**.

# Why VectorFlow AI?

Creating production-ready Illustrator files by hand is repetitive, time-consuming, and difficult to scale:

- Import images
- Trace artwork
- Clean up vectors
- Export AI files
- Check production quality

VectorFlow AI automates this entire workflow, allowing designers and manufacturers to focus on creative work instead of repetitive production tasks.

---

# Key Features

### AI-powered Workflow Automation

Automatically launches and controls Adobe Illustrator to generate editable vector artwork.

### Intelligent Image Analysis

Analyzes input images before vector production without manual preparation.

### Batch Processing

Process multiple images automatically without repetitive manual work.

### Editable Illustrator Files

Outputs fully editable `.ai` files instead of flattened graphics.


### AI Quality Classification

Automatically classifies production results into:

🟢 Class A — Ready for production

🟡 Class B — Manual review recommended

🔴 Class C — AI could not generate a reliable result

### Local Processing

Your artwork never leaves your computer. 

Only AI requests are sent using your own Gemini API key.

### Bring Your Own Gemini API Key

Use your own Gemini API key. No VectorFlow AI account or subscription is required.

### Intelligent Proportional Scaling

Users can specify target dimensions.

VectorFlow AI preserves the original aspect ratio to prevent distortion.

### Bilingual Interface

English

中文

Switch anytime from the Settings window.

# Installation

1. Download the latest release from the [Releases](https://github.com/Miliinthesky/vectorflow/releases/tag/v1.0.0) page.
2. Open **VectorFlow AI.app**.
3. Enter your Gemini API key.
4. Select your input image.
5. Click **Start** to begin the automated workflow.
