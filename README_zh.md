# VectorFlow AI - 告别手动描摹，专注创意设计。

🌐 [English](README.md) | **简体中文**

### 面向 Adobe Illustrator 的 AI 智能矢量生产工作流

*不仅仅是矢量化，更让整个 Illustrator 工作流程实现 AI 自动化。*

VectorFlow AI 是一款基于 AI 的桌面应用程序，可自动完成 Adobe Illustrator 中的整个矢量生产工作流。

它能够分析图片、生成可编辑的 Adobe Illustrator（`.ai`）文件、自动评估输出质量，并帮助设计师和制造商高效完成重复性的生产任务。

# 系统要求

使用 VectorFlow AI 前，请确保您的电脑满足以下要求：

✅ Adobe Illustrator（必需）

✅ Gemini API Key

✅ macOS

整个处理过程中**无需手动操作 Adobe Illustrator**。

VectorFlow AI 会自动启动并控制 Illustrator，完成整个生产流程。

# 使用流程

首次启动 VectorFlow AI 时，会自动弹出欢迎窗口，请输入您的 Gemini API Key。

<img width="448" height="232" alt="Welcome En:CN" src="https://github.com/user-attachments/assets/1948be9f-d085-4bc6-a462-a9e747f0c279" />

将需要处理的 PNG 或 JPG 图片放入一个文件夹，选择输入文件夹和输出文件夹，然后点击 **"Start Smart Processing"**。

VectorFlow AI 将自动启动 Adobe Illustrator，并完成后续全部处理流程，无需任何人工干预。

<img width="898" height="681" alt="UI En" src="https://github.com/user-attachments/assets/95c131c7-fb7a-4406-b349-d57363cd5304" />

点击 **"Settings"** 可调整输出尺寸。软件会根据原始图片自动保持宽高比例，避免图像变形。

您还可以在此切换中文或英文界面，完成设置后点击 **"Save"** 保存。

# 为什么选择 VectorFlow AI？

手动制作符合生产要求的 Illustrator 文件通常需要重复完成以下工作：

- 导入图片
- 图像描摹（Trace）
- 清理矢量路径
- 导出 AI 文件
- 检查生产质量

VectorFlow AI 将整个流程自动化，让设计师和制造商能够把更多时间投入到创意工作，而不是重复、繁琐的生产流程。

# 核心功能

### AI 智能工作流自动化

自动启动并控制 Adobe Illustrator，生成可编辑的矢量图稿。

### 智能图像分析

在矢量生产前自动分析输入图片，无需手动预处理。

### 批量处理

支持批量处理多张图片，无需重复进行人工操作。

### 可编辑的 Illustrator 文件

输出完全可编辑的 `.ai` 文件，而不是不可编辑的位图或扁平化图形。

### AI 智能质量分类

自动对生成结果进行分类：

🟢 A 类 —— 可直接投入生产

🟡 B 类 —— 建议人工复核

🔴 C 类 —— AI 无法生成可靠结果

### 本地处理

您的图片始终保存在本地电脑。

只有 AI 分析请求会通过您自己的 Gemini API Key 发送至 Gemini 服务。

### 使用您自己的 Gemini API Key

无需注册 VectorFlow AI 账号，也无需订阅任何服务。

只需使用您自己的 Gemini API Key 即可开始使用。

### 智能等比例缩放

支持自定义目标尺寸。

VectorFlow AI 会自动保持原始图片宽高比例，避免图像拉伸变形。

### 双语界面

- English
- 中文

可随时在 **Settings** 窗口中切换界面语言。

# 安装

1. 从 [Releases](https://github.com/Miliinthesky/vectorflow/releases/tag/v1.0.0) 页面下载最新版本。
2. 打开 **VectorFlow AI.app**。
3. 输入您的 Gemini API Key。
4. 选择输入图片。
5. 点击 **Start** 开始自动处理工作流。
